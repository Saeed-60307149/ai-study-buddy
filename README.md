# ai-study-assistant

## Description
This app helps students study by summarizing notes and generating quiz questions using AI.

## Features
- Upload or paste notes
- AI summary
- AI quiz generation

## Tech Stack
- Backend: Python (Flask)
- AI: Gemini API
- Database: MongoDB Atlas
- Deployment: Render
- CI/CD: GitHub Actions
